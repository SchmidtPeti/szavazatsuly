# SzavazatSúly

Elérhetőség: https://SchmidtPeti.github.io/szavazatsuly/

A SzavazatSúly egy rövid, mobilra is optimalizált információs oldal, ahol irányítószám alapján meg lehet nézni, hogy a 2022-es országgyűlési választáson mekkora különbséggel dőlt el az adott egyéni választókerület. Az oldal azt mutatja meg, hány szavazaton múlt a győzelem, és segít érzékeltetni, hogy az egyéni mozgósítás mekkora hatással lehet egy szoros körzetben.
